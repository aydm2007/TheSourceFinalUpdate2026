# Nexus Memory – دليل الاستخدام الرئيسي

> **الهدف:** حفظ واسترجاع المعرفة الحيوية بين جلسات TheSource بطريقة سيادية ومؤمنة.

## محتويات الدليل
- **PROTOCOLS.md** – بروتوكولات الأمن، الفهارس، وZodSchema.
- **CLOUDOPS.md** – دمج مع CloudOps 4.7 (IaC, مراقبة, أسرار).
- **USAGE.md** – كيفية استدعاء الوكلاء الداخليين وتخزين نتائج "الإسـراب".
- **SKILL.md** – ملف تعريف المهارة (يُستبدل بملف موجز).

---

### نظرة سريعة
- جميع عمليات **القراءة/الكتابة** يجب أن تمر عبر **ZodSchema** للتحقق من صحة المدخلات.
- لا تُحفظ أي مفاتيح، كلمات مرور أو بيانات شخصية.
- كل تعديل يُسجل في **shadow_ledger.jsonl** مع توقيت ودقة.
- يُفضَّل تقسيم أي طلب تحليل كبير إلى أجزاء (Chunk‑by‑Chunk) لتفادي حد الـ tokens في نماذج Gemini‑Flash 3.
