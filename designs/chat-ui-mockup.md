# Chat UI Mockup (Dark Mode)

### Typography
- **Font Family**: "Inter", system UI fallback.
- **Base Size**: 16px (desktop), 14px (mobile).
- **Message Text**: 15px, line‑height 1.4.
- **Headers**: 20px bold for chat title, 14px for timestamps.

---
### Animations & Interactions
1. **Message Fade‑In** – New messages fade from 0% to 100% opacity over 200 ms.
2. **Slide‑Up Input** – Input bar slides up when focused on mobile (keyboard appearance).
3. **Button Ripple** – Material‑style ripple on send/emoji buttons.
4. **Hover Elevation** – Chat list items raise slightly with box‑shadow on hover.
5. **Typing Dots** – Three pulsing dots with 0.6 s loop.

---
### Responsive Breakpoints
| Breakpoint | Width Range | Adjustments |
|------------|------------|------------|
| Desktop    | ≥ 1024px   | Sidebar visible, chat list width 260 px. |
| Tablet     | 768‑1023px | Sidebar collapsible, toggle button in header. |
| Mobile     | < 768px    | Sidebar hidden, full‑screen chat view, bottom nav for switching chats. |

---
### Accessibility
- **Contrast**: Minimum 4.5:1 for text vs. background.
- **ARIA Labels**: All interactive elements have descriptive `aria-label`.
- **Keyboard Navigation**: Tab order logical, focus outlines visible.
- **Reduced Motion**: Respect `prefers-reduced‑motion` – disable non‑essential animations.
